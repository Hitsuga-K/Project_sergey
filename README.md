# NEON JUMP 🚀

**NEON JUMP** — это многопользовательская 3D-игра в стиле киберпанк. Игроки управляют неоновыми кубами на огромной платформе, соревнуются в высоте прыжков, зарабатывают монеты и сражаются за первое место в рейтинге. В игре реализована полная авторизация, чат, обновление персонажа в реальном времени и динамическая корона лидера.

## 🎮 Особенности

- 🔐 **Регистрация / Вход** — безопасное хранение паролей (bcrypt) и JWT-токены.
- 🕹️ **3D-движок на Three.js** — неоновое освещение, анимированные персонажи, свободное перемещение.
- 🦘 **Физика прыжков** — чем выше прыжок, тем больше монет получает игрок.
- 👑 **Корона лидера** — игрок с максимальным количеством монет отображается с золотой короной в реальном времени.
- 🗣️ **Чат** — общение всех игроков внутри игры.
- 🧮 **Таблица лидеров** — топ-10 игроков по количеству монет (обновляется автоматически).
- ⚡ **Улучшение прыжка** — за монеты можно увеличить силу прыжка (и, соответственно, заработок).

## 📦 Технологический стек

| Компонент       | Технология                                 |
|----------------|--------------------------------------------|
| Backend        | Node.js + Express 5                        |
| База данных    | PostgreSQL (с SSL-соединением)            |
| Аутентификация | JWT + bcryptjs                             |
| Realtime       | Socket.IO                                  |
| 3D графика     | Three.js (r128)                            |
| Стили          | CSS (неоновый киберпанк)                   |
