# 📁 public - Клиентская часть игры

Эта папка содержит все статические файлы, которые отправляются клиенту для запуска игры в браузере.

## 📁 Структура папки
public/
├── 📁 js/ # Клиентские JavaScript файлы
│ ├── 📄 auth.js # Система аутентификации
│ ├── 📄 config.js # Глобальные настройки и переменные
│ ├── 📄 game.js # Игровая логика (Three.js)
│ ├── 📄 socket.js # WebSocket обработчики
│ └── 📄 ui.js # UI компоненты и мобильные контролы
├── 📄 index.html # Главная страница
└── 📄 style.css # Стилизация интерфейса

text

## 📄 index.html

Главный HTML файл, который:
- Подключает все необходимые скрипты и стили
- Определяет структуру UI
- Инициализирует игру

## 🎨 style.css

Основные стили проекта:
- **Неоновый дизайн** с использованием цветов: `#00ffff` (циан) и `#ff00ff` (магента)
- **Адаптивность** для мобильных и десктопных устройств
- **Анимации** для UI элементов
- **Мобильные контролы** (джойстик и кнопка прыжка)

### Медиа-запросы

- `max-width: 768px` - Мобильные устройства
  - Показывает джойстик и кнопку прыжка
  - Уменьшает размеры UI элементов
  - Адаптирует чат и таблицу лидеров

- `min-width: 769px` - Десктопы
  - Скрывает мобильные контролы
  - Показывает полноценный интерфейс

## 📁 js/ - Клиентская логика

Подробное описание каждого файла в [js/README.md](js/README.md)

## 🚀 Как это работает

1. Пользователь открывает `index.html`
2. Загружаются все стили и скрипты
3. Показывается форма аутентификации
4. После успешного входа:
   - Инициализируется Three.js сцена
   - Устанавливается WebSocket соединение
   - Создается 3D мир
   - Добавляются игроки
   - Начинается игровой процесс

## 📱 Мобильная поддержка

Игра полностью адаптирована для мобильных устройств:
- **Сенсорный джойстик** для движения
- **Большая кнопка** для прыжка
- **Адаптивные размеры** UI
- **Оптимизированная производительность**

## 🔧 Настройка

Для изменения внешнего вида игры:
1. Цвета и шрифты в `style.css`
2. 3D сцена и физика в `js/game.js`
3. UI элементы в `index.html`

Для изменения игровой логики:
1. Правила сбора монет - `js/game.js`
2. Система улучшений - `js/ui.js` и `js/socket.js`
3. Сетевое взаимодействие - `js/socket.js`
