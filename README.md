## 🌟 Обзор проекта

**NEON JUMP** - это захватывающая многопользовательская 3D игра, где игроки соревнуются за сбор монет, улучшение характеристик и место в таблице лидеров. Игра выполнена в неоновом стиле с использованием Three.js для 3D графики.

### ✨ Особенности

- 🎮 Многопользовательская игра в реальном времени
- 💰 Система сбора монет при прыжках
- 🏆 Таблица лидеров
- 👑 Система "короны" для лучшего игрока
- 📱 Адаптивный интерфейс для мобильных устройств
- 💬 Встроенный чат
- 🚀 Система улучшения прыжка
- 🎨 Неоновая графика и эффекты

## 🛠 Технологии

### Frontend
- **HTML5/CSS3** - Структура и стилизация
- **JavaScript (ES6)** - Логика игры
- **Three.js r128** - 3D рендеринг
- **Socket.IO Client** - Реальное время

### Backend
- **Node.js** - Серверная платформа
- **Express.js** - Веб-фреймворк
- **Socket.IO** - WebSockets для мультиплеера
- **PostgreSQL** - База данных
- **JWT** - Аутентификация
- **bcryptjs** - Хеширование паролей

## 🚀 Установка и запуск

- Node.js (версия 18.x или выше)

🎮 Геймплей
Основные механики
Движение: WASD или джойстик на мобильных устройствах

Прыжок: Пробел или кнопка прыжка на мобильных

Сбор монет: При приземлении после прыжка

Улучшение: Увеличивайте силу прыжка за монеты

Система монет
За каждый прыжок вы получаете монеты

Количество монет зависит от высоты прыжка

Монеты можно тратить на улучшение прыжка

Таблица лидеров
Отображает топ-10 игроков по количеству монет

Обновляется в реальном времени

Лучший игрок получает корону в игре
