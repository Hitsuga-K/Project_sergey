# 📁 js - Клиентские скрипты

Эта папка содержит всю клиентскую логику игры NEON JUMP.

## 📄 Содержание

- [auth.js](#-authjs) - Аутентификация
- [config.js](#-configjs) - Конфигурация
- [game.js](#-gamejs) - Игровая логика
- [socket.js](#-socketjs) - Веб-сокеты
- [ui.js](#-uijs) - UI и контролы

## 🔐 auth.js

Управляет аутентификацией пользователей.

### Основные функции:
- `registerBtn.onclick` - Обработка регистрации
- `loginBtn.onclick` - Обработка входа
- `startGame()` - Инициализация игры после входа

### Поток работы:
1. Пользователь вводит логин/пароль
2. Отправка запроса на `/api/register` или `/api/login`
3. Получение JWT токена
4. Запуск игры с переданным токеном
